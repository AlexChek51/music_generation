# Генерация музыки с помощью LSTM на основе MIDI-файлов

## Описание:
Этот проект демонстрирует создание и использование модели генерации музыки на основе MIDI-файлов с помощью рекуррентных нейронных сетей (LSTM). Проект включает в себя:
- Загрузку и подготовку MIDI-данных.
- Извлечение и преобразование нот и аккордов из MIDI-файлов.
- Построение и обучение модели LSTM для генерации музыкальных последовательностей.
- Генерацию новых музыкальных произведений на основе обученной модели.
